You want notifications for specific job postings popping up on your favourite companies' career pages.

But you want to avoid the clunky mess of dealing with job search engines.

You also don't have the time or discipline to manually check all those career pages.

joblert.me service solves these pain points by automating this process for you. This is acheived by searching all your listed career pages every hour and checking to see if any new postings are a match with your custom queries.

Simple yet effective.

How To: Write More Advanced Queries


By default, keyword entries are separated by spaces. The query will only pull up job postings containing all of the keywords although they don't nescessarily need to be side-by-side in the posting's text.

User-defined search phrase

Data Scientist
Resulting query
successful posting will need to contain Data and Scientist




Using parantheses in conjunction with special keyword will create alternate queries. If any of the queries matches, the posting will be sent.


User-defined search phrase
SQL (Python OR R)
Resulting queries
successful posting will need to contain SQL and either Python or R




Using single quotes '' will only return postings that contain the exact string match within the single quotes.


User-defined search phrase
'Data (Scientist OR Analyst)'
Resulting queries
successful posting will need to contain either Data Scientist or Data Scientist
